Built with:

- [NextJS](https://nextjs.org/)
- [Tailwindcss](https://tailwindcss.com/)

## Getting Started

1. Install dependencies

   ```bash
   npm install
   ```

2. Fire up the server and watch files

   ```bash
   npm run dev
   ```

## Deployment

1. Install the Now CLI

   ```bash
   npm run build
   ```

2. Deploy!

   ```bash
   npm start
   ```
